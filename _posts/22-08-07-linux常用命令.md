## 防火墙相关的
Debian 和 Ubuntu
```bash
# debian和ubuntu可以通过如下的方式安装ufw
apt install ufw 
# 查询端口的开放的情况
ufw status
# 开放某一个端口
ufw allow 22/tcp
```
## 关机
```sh
# 立刻关机,需要root用户
shutdown -h now

# 10分钟后自动关机
shutdown -h 10
```
